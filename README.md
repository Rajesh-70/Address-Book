# Address-Book
Address Book is a console-based application in C used to store and manage contacts such as name, phone number, and email. It supports add, view, search, edit, and delete operations. The project demonstrates core C concepts like structures, pointers, and file handling for persistent data storage.

# Overview
This is a console-based Address Book application developed in C.
It allows users to manage contacts efficiently using file handling.

## Features
- Create new contacts
- View all contacts
- Search contacts
- Edit contact details
- Delete contacts
- Data stored permanently using files

## Concepts Used
- File I/O (fopen, fread, fwrite)
- Structures
- Pointers
- Modular programming
- Header files

## Tools
- C Language
- GCC Compiler
- Linux / Windows

```bash
gcc *.c -o addressbook
